# cn-pc-journal

本仓库用于归档与发布多个独立 HTML 页面，内容涵盖各类静态网页资源。仓库不针对任何特定域名或网站，仅作为页面文件的存储与展示空间。

## 项目简介

- 本仓库主要存放独立 HTML 页面文件，每个页面均为自包含的静态网页。
- 页面主题多样，不预设使用场景或受众。
- 所有页面均可在浏览器中直接打开使用，无需额外依赖。

## 目录说明

```
.
├── index.html          # 仓库首页，列出所有页面索引（如存在）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 页面共用资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下每个 HTML 文件均为一个独立页面。
- `assets/` 目录存放页面可能调用的共用资源文件。
- 索引页面（`index.html`）可选，用于快速导航。

## 页面归档说明

- 每个页面文件均保持原始结构与内容，不进行修改或重定向。
- 归档页面可能包含历史版本，文件名或目录内可附带日期标识。
- 页面内容不代表本仓库维护者的立场或观点。

## 维护说明

- 本仓库不定期更新，新增页面或修正已有页面。
- 欢迎通过 Issues 提交问题或建议，但请勿包含具体域名、网址或营销内容。
- Pull Request 需遵循以下规则：
  - 新增页面请放入 `pages/` 目录，并确保为独立 HTML 文件。
  - 不引入外部资源或第三方服务链接。
  - 不添加任何诱导点击或推广内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `pages/` 下的任意 HTML 文件。
3. 或通过 GitHub Pages 部署后在线访问（需自行配置）。
